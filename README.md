# 1_artificial-neural-networks

Recursos y notas acerca del entendimiento del funcionamiento de las redes neuronales desde el punto de vista de implementación y operaciones matemáticas, junto con implementaciones en scripts de Python evitando utilizar librerías externas que hagan este trabajo para comprender su funcionamiento en el código

Caben destacar implementaciones para:
- Alimentación hacia delante
- Algoritmo de descenso del gradiente
- Aprendizaje adaptativo con Adam
- Resolución de problemas de regresión y clasificación con sets de datos etiquetados
- Tips y otros
